Component Architecture

1. Document Ingestion Layer
PyMuPDF (fitz) processes PDF documents from the /docs directory, extracting text content while preserving document structure and metadata.
  • Handles multiple PDF files concurrently
  • Preserves source document tracking
  • Error handling for corrupt or problematic files
2. Text Processing Pipeline
RecursiveCharacterTextSplitter intelligently segments documents into manageable chunks.
  • Chunk size: 500 characters
  • Overlap: 100 characters for context preservation
  • Maintains chunk-to-document mapping
3. Embedding Engine
SentenceTransformer (all-MiniLM-L6-v2) converts text chunks into semantic vectors.
  • 384-dimensional embeddings
  • Optimized for semantic similarity
  • Batch processing capability
